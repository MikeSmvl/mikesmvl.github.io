# mikaelsamvelian.com

Personal portfolio website built as an interactive 3D credit card. The front face shows name, title, and monogram; the back face has links, a featured project, and location. The card flips on click/scroll/swipe and tilts with mouse movement.

GitHub Pages serves from `master`. Pushing to `master` deploys to [www.mikaelsamvelian.com](https://www.mikaelsamvelian.com).
